# selenium
什么是 Selenium
Selenium 是一个用于<b>测试 Web 应用程序</b>的自动化工具。
Selenium 测试直接运行在浏览器中，就像真正的用户在操作一样。
主要功能\
<b>测试系统功能</b>\
创建回归测试检验软件功能和用户需求。\
<b>测试与浏览器的兼容性</b>\
测试应用程序看是否能够很好得工作在不同浏览器和操作系统之上。\
